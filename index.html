Les commandes Git

Commande Description
git init Initialise le depôt Git
git status Affiche l'état de l'index Git
git add nom_de_fichier Ajoute à l'index Git le fichier nom_de_fichier
git commit -m "mon_message" Ajoute une trace validée de modifications effectuées sur le projet dans l'historique du dépôt Git
git remote add origin url_depot_externe Ajoute un lien vers un depôt externe url_depot_externe
git push -u origin master Envoie les modifications de la branche principale vers le depôt externe lié
